# Job-Board-Application

This is a job board app built with React and Material-UI. It allows users to view job listings, apply for jobs, and manage their applications.

## Features

- View a list of job postings with details such as job title, location, and job description.
- Apply for jobs directly from the app.
- Manage applied jobs and saved jobs.
- Responsive design for mobile and desktop devices.

## Installation

1. Install dependencies:
    npm install
2. Start the development server:
    npm start

## Usage
1. Browse through the list of job postings on the home page.
2. Click on a job listing to view detailed information.
3. Apply for jobs by clicking the "Apply" button.
4. Save jobs for later viewing by clicking the "Save" button.

## Technologies Used
React.js
Material-UI
TypeScript
JSON Server (for mock data)
